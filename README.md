вңЁ Features - рҹ”җ **JWT Authentication** вҖ” Secure login/register with bcrypt password hashing - рҹ’¬ **RAG-Powered Chat** вҖ” Answers grounded in real medical documents - рҹ—„пёҸ **Chat History** вҖ” Persistent chat history per user (PostgreSQL) - рҹҺЁ **Dark Theme UI** вҖ” Professional medical-grade interface (React) - рҹ“ҡ **8 Medical Knowledge PDFs** вҖ” Covering diseases, drugs, diagnostics, emergencies & more - рҹҗі **Docker Ready** вҖ” One command deployment with docker-compose - вҡЎ **Groq LLM** вҖ” Ultra-fast inference with LLaMA 3.3 70B - рҹ”Қ **Semantic Search** вҖ” ChromaDB vector store with cosine similarity --- ##
